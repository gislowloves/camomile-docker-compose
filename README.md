🌸 **Ромашковая Долина**

Интерактивная текстовая стратегия на C++, в которой вы берете на себя роль старейшины знаменитой деревни Смешариков. Принимайте важные решения, следите за настроением жителей и сохраняйте результаты в базу данных Redis.

**Запуск**

Для запуска проекта вам понадобится только Docker и Docker Compose.

Сборка проекта:

```Bash
sudo docker compose build
```

Запуск игры (интерактивный режим):

```Bash
sudo docker compose run --rm app
```
